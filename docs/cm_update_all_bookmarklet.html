CM Update All bookmarklet

One-click tool for Benchling's Configuration Migration import: flips every Create action to Update of the same-named object, so you don't have to change hundreds of dropdowns by hand. Full instructions: Bulk "Update" Selection for Configuration Migration Import.

Install (once)

Drag each button below onto your browser's bookmarks bar (don't click them here — they only act on a Benchling import page):

CM Update All v1

Flips every "Create" to "Update <same-named object>".

CM Update All v1 (dry run)

Shows what would be selected for each row; changes nothing.

Use

  1. In Benchling, go to Configuration migration → Import, upload your .dat file, and continue to the Validate plan page.
  2. Open the SCHEMAS tab (sections are expanded by default).
  3. Click the bookmarklet. A status box appears in the corner of the page with progress, a Stop button, and a list of any rows needing manual attention. Running the dry-run variant first is a good habit.

The bookmarklet runs entirely in your browser tab: it makes no network requests and touches no credentials — it only performs the same clicks you would make yourself. Nothing is submitted to Benchling until you click Continue in the wizard. The full code is the bookmark's URL; the readable source is cm_update_all.js.

Updates: the code lives inside your bookmark, and the status box shows its version (currently v1). When a newer version is released (for example after a Benchling UI change), delete the old bookmarks and drag the new ones from the latest version of this page.
